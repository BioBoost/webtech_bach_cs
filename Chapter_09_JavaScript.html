Introduction

A scripting language

  • JavaScript is the most popular scripting language in the world. It is the standard language of the web, and widely used in servers, desktops, and mobile phones.
  • JavaScript is a Scripting Language
    • A scripting language is a lightweight programming language that supports the writing of scripts.
    • JavaScript was designed to add interactivity to HTML pages.
    • JavaScript is programming code that can be inserted into HTML pages to be executed by web browser.

Running JavaScript

  • All major web browsers contain JavaScript interpreters, which process the commands written in JavaScript.
  • Before you can run code examples with JavaScript on your computer, you may need to change your browser’s security settings.
    • IE9 prevents scripts on the local computer from running by default
    • Firefox, Chrome, Opera, Safari, ... browsers have JavaScript enabled by default.

Capabilities of JavaScript

  • JavaScript can write directly into the HTML output stream.
  • JavaScript can change the content of HTML elements.
  • JavaScript can react to events.
  • JavaScript can manipulate HTML styles.
  • JavaScript can test input, and alert about errors.
  • ...

Idempotent

  • The increment and decrement operators are not idempotent and can have side-effects if not used with caution.
    • Idempotent operations always give the same result, no matter how many times they are executed.

    //...

  • Results are different because of idempotency in combination with lazy evaluation.

JavaScript Relational Operators

JavaScript Relational Operators

  • As mentioned before, JavaScript automatically converts between types when needed.
  • This results in the following evaluation being evaluated as true:

<script type="text/javascript">
if (75 == "75")
    document.writeln("'75' equals 75");
else
    document.writeln("'75' does not equals 75");
</script>
  • This behaviour can be circumvented using the strict comparison operators (=== and !==).

<script type="text/javascript">
if (75 === "75")
    document.writeln("'75' strict equals 75");
else
    document.writeln("'75' does not strict equals 75");
</script>

JavaScript Logical Operators

  • Logical operators can be used to form complex conditions by combining simple conditions
    • && (logical AND)
    • || (logical OR)
    • ! (logical NOT, also called logical negation)

JavaScript Logical Operators

  • JavaScript evaluates to false or true all expressions that include relational operators, equality operators and/or logical operators.
    • In JavaScript, any nonzero numeric value in a condition evaluates to true, and 0 evaluates to false.
    • Any string containing one or more characters evaluates to true, and the empty string "" evaluates to false.
    • The value null and variables that have been declared but not initialized are considered to be false.
    • All objects are considered to be true.

JavaScript functions

Function invocation

  • Functions are invoked by writing the name of the function, followed by a left parenthesis, followed by a comma-separated list of zero or more arguments, followed by a right parenthesis.

JavaScript Arrays

  • JavaScript arrays are "dynamic" entities that can change size after they are created.
  • An array is a group of memory locations.
    • All have the same name and normally are of the same type (although this attribute is not required in JavaScript)
  • We may refer to any one of the elements by giving the array's name followed by the position number of the element in square brackets [].
    • The first element in every array is the zeroth element.

JavaScript Arrays

  • JavaScript arrays are Array objects.
    • Every array in JavaScript knows its own length, which it stores in its length attribute and can be found with the expression arrayname.length
  • You use the new operator to create an array and to specify the number of elements in an array.
    • The new operator creates an object as the script executes by obtaining enough memory to store an object of the type specified to the right of new.

Multidimensional arrays

  • In general, an array with m rows and n columns is called an m-by-n array.
  • Specify the two indices to identify a particular two-dimensional multidimensional array element.
    • By convention, the first identifies the element's row, and the second identifies the element's column.

Multidimensional arrays

  • Two-dimensional array elements are accessed using an element name of the form a[row][column].

Multidimensional arrays

  • Are actually arrays of arrays

<script type="text/javascript">
var myArray = new Array(3);     // 3 rows
myArray[0] = new Array(10);    // 10 columns
myArray[1] = new Array(10);    // 10 columns
myArray[2] = new Array(10);    // 10 columns

for (var row = 0; row < myArray.length; row++)
{
    for (var col = 0; col < myArray[row].length; col++)
    {
        myArray[row][col] = row * col;
    }
}
</script>

Jagged arrays

  • Sub arrays are not required to have the same length.

    for (var col = 0; col < myArray[row].length; col++) {

Reference parameters

Reference parameters

  • There are two ways to pass arguments to functions (or methods)
    • Pass-by-value
      • a copy of the argument's value is made and is passed to the called function
    • Pass-by-reference
      • The caller gives the called function access to the caller’s data and allows the called function to modify the data if it so chooses.
      • Can improve performance because it can eliminate the overhead of copying large amounts of data, but it can weaken security because the called function can access the caller's data.

Reference parameters

  • In JavaScript, numbers, boolean values and strings are passed to functions by value.
  • All objects are passed to functions by reference.
  • With pass-by-value, changes to the copy of the value received by the called function do not effect the original variable's value in the calling function.
  • When information is returned from a function via a return statement, numbers and boolean values are returned by value and objects are returned by reference.

Passing arrays to functions

  • To pass an array as an argument to a function
    • Specify the array's name (a reference to the array) without brackets
    • Although entire arrays are passed by reference, individual numeric and boolean array elements are passed by value exactly as simple numeric and boolean variables are passed
      • Such simple single pieces of data are called scalars, or scalar quantities.
      • To pass an array element to a function, use the indexed name of the element as an argument in the function call.
