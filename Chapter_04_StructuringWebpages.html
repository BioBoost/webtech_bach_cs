Floating elements

  • The float property allows you to take an element in normal flow and place it as far to the left or right of the containing element as possible.
  • Anything else that sits inside the containing element will flow around the element that is floated.
  • You should also use the width property to indicate how wide the floated element should be.

Inlining the list items

Switching between inline and block

display

  • The display property allows you to change the elements display style.
  • Possible values
    • inline
    • block
    • inline-block: causes a block-level element to flow like an inline element, while retaining other features of a block-level element.
    • none: hides an element from the page (the element acts as though it is not on the page at all).

The clear property

  • The clear property allows you to say that no element (within the same containing element) should touch the left or right-hand sides of a box.
  • Possible values:
    • left: left-hand side of the box should not touch any other elements appearing in the same containing element.
    • right: right-hand side of the box will not touch elements appearing in the same containing element.
    • both: Neither left nor right-hand sides of the box will touch elements appearing in the same containing element.
    • none: elements can touch either side.
